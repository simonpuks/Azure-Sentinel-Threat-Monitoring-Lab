**Azure Sentinel Threat Monitoring and Detection Lab**
**Project Overview**

This project demonstrates the deployment of a cloud-based Security Operations Centre (SOC) environment using Microsoft Azure and Microsoft Sentinel.

The objective was to simulate a real-world security monitoring setup capable of collecting security logs, detecting attacks, investigating incidents, and visualising security events through dashboards and workbooks.

## Project Objectives

- Deploy an Azure Virtual Machine  
- Configure Log Analytics Workspace  
- Connect Microsoft Sentinel  
- Collect Windows Security Events  
- Monitor RDP and SSH activity  
- Detect brute-force attacks  
- Perform threat hunting using KQL  
- Create Sentinel Analytics Rules  
- Build security dashboards and workbooks  
- Investigate security incidents
